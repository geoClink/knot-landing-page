# Knot Landing Page

Marketing landing page for [Knot](https://knotsystem.com) — the operating system for local business.

## Stack

- **HTML / CSS / JavaScript** — no framework, vanilla web
- **Tailwind CSS v4** — utility-first styling
- **Vite** — local dev server and production build tool

## Getting Started

**Prerequisites:** Node.js installed on your machine.

```bash
# Install dependencies
npm install

# Start the local dev server (hot-reloads on save)
npm run dev
```

Open `http://localhost:5173` in your browser.

## Project Structure

```
knot-landing-page/
├── index.html          # Main page — all sections live here
├── src/
│   ├── index.css       # Global styles and CSS variables
│   └── main.js         # JavaScript (interactions, animations)
├── public/
│   ├── images/         # Static images served as-is
│   ├── icons.svg       # SVG icon sprite
│   └── og-image.png    # Open Graph / social share image
└── dist/               # Built output (generated by npm run build)
```

## Common Commands

| Command | What it does |
|---|---|
| `npm run dev` | Start local dev server at `localhost:5173` |
| `npm run build` | Build for production into `dist/` |
| `npm run preview` | Preview the production build locally |

## Deployment

Run `npm run build` to generate the `dist/` folder, then deploy that folder to your hosting provider.
